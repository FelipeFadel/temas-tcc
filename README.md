## Git Lfs

A ideia é que em vez de salvar o arquivo grande direto no Git, você salva apenas um ponteiro em texto, o arquivo real fica em um armazenamento separado.

Um git lfs tem a seguinte aparência: 

`version https://git-lfs.github.com/spec/v1
oid sha256:4cac19622fc3ada9c0fdeadb33f88f367b541f38b89102a3f1261ac81fd5bcb5
size 84977953
`
